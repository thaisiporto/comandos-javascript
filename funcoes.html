Funções

Funções (function): comandos executados somente quando eventos DOM forem ativados

Funções devem receber nomes:

function nomeDaAcao(parametros adicionais) { bloco }

Bloco: conjunto de linhas que formam um comando, delimitado por chaves {}.

Adicionando Eventos

Funções podem ser adicionadas no html ou no script.

No html:

<p id="cliqueAqui"
onclick="clicar()">Clique aqui!</p>

<script>
  function clicar() {  
    var clck = window.document.
querySelector("p#cliqueAqui"); 
    clck.innerText = "Clicou!"; 
    clck.style.background = "#7c9cc7"; 
  } 
 </script>

Clique aqui!

No script:

<p id="cliqueAqui">Clique aqui!</p>

<script>  
  var clck = window.document.
querySelector("p#cliqueAqui"); 
  clck.addEventListener("click", clicar); 
  function clicar() {  
    clck.innerText = "Clicou!"; 
    clck.style.background = "#ffc0cb"; 
  } 
 </script>
