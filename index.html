Comandos Básicos

Alguns comandos em Javascript conforme vão sendo aprendidos.

window.alert("Essa é uma mensagem de Alerta!");

Esse comando abre uma janela de alerta na página.

window.alert("Essa é uma mensagem de Confirmaçao!");

Esse comando abre uma janela de confirmação na página.

window.prompt("Qual é seu nome?");

Esse comando abre uma janela com um input para entrada de texto.

varName.style.color = "#3a4669";

Muda a cor do texto do objeto.

Esse texto vai mudar de cor!

varName.style.background = "#3a4669";

Muda a cor de fundo do elemento.

O fundo desse texto vai mudar de cor!

varName.innerText = "Mudou!";

Adiciona ou muda o texto do objeto.

Esse texto vai mudar!

varName.toUpperCase();

varName.toLowerCase();

Transforma o texto em maiúsculas ou minúsculas (respectivamente).

Esse Texto Vai Mudar!

varName.length;

Conta quantos caracteres o texto possui.

Quantas letras palavra tem?
