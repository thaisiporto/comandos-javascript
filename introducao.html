Estudos em Javascript

Introdução

O que é Javascript?

É a linguagem de programação mais popular do mundo, é considerada uma tecnologia fundamental, juntamente com HTML e CSS, da World Wide Web.

O Javascript permite maior interatividade em uma página, como sliders, alertas, pop-ups e interações de clique.

Fonte: Roadmaps

Javascript é uma linguagem de programação considerada não compilada, ou interpretada, ou seja, o navegador executa diretamente os scripts linha-a-linha.

O javascript é "case sensitive", as letras maiúsculas e minúsculas fazem diferença.

Como adicionar scripts

Os scripts podem ser adicionados em um documento HTML, dentro tag <script> </script> e localizada acima da tag de fechamento </body>.

O Javascript também pode ser adicionado em um documento separado e anexado acima da tag de fechamento </body>:

<script src="scripts/my-scripts.js"> </script>

Tipos de Dados / Valores

  • Number (Número): Números inteiros ou com casas decimais.
  • String: Cadeia/agrupamento de caracteres (letras, números, símbolos, espaços, etc). Devem estar entre aspas simples, duplas ou crases.
  • Boolean (Booleano): Tipo de dado com apenas dois valores possíveis: verdadeiro (true) ou falso (false).
  • null (Nulo): Aparecendo sempre em letras minúsculas, representa um valor inexistente.
  • undefined (Indefinido): Também escrito em letras minúsculas, aparece quando uma variável não recebeu um valor.
  • Symbol (Símbolo): É um recurso mais recente no ECMAScript, utilizado em códigos mais avançados. Ele armazena um valor único e imutável.
  • Object (Objeto): Diferente dos tipos de dados anteriores, o object não é um tipo primitivo. Ele pode ser descrito como uma coleção de dados e propriedades.

Variáveis

São objetos que usados para rotular e armazenar dados e valores.

As variáveis precisam ser declaradas com uma palavra-chave, um identificador (rótulo) e o valor atribuído.

var rótulo = "valor atribuído";

As palavras-chaves utilizadas para declarar variáveis são:

  • var: É a primeira e mais antiga palavra-chave utilizada para declarar variáveis, ela existe desde o início da linguagem JavaScript. Os valores em uma variável do tipo var podem ser atualizados e funcionam em escopo global.
  • let: A palavra-chave let foi introduzida na versão ECMAScript 6, em 2015. Ela funciona como a variável var, podendo ser atualizada, mas como escopo limitado ao bloco.
  • const: Também foi introduzida na versão ECMAScript 6. Const significa constante, os valores armazenados nessa variável não podem ser atualizadas ou redeclaradas.

Variáveis precisam de identificadores:

  • Podem começar com letras, $ ou _.
  • É possível utilizar letras e números.
  • Não podem começar com números.
  • É possível utilizar acentos e símbolos.
  • Não podem conter espaços.
  • Não podem conter palavras reservadas, como function, alert, etc.
  • Deve-se usar nomes coerentes.
  • Maiúsculas e minúsculas fazem diferença.
