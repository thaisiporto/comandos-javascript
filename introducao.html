Introdução

O que é Javascript?

É a linguagem de programação mais popular do mundo, é considerada uma tecnologia fundamental, juntamente com HTML e CSS, da World Wide Web.

O Javascript permite maior interatividade em uma página, como sliders, alertas, pop-ups e interações de clique.

Fonte: Roadmaps

Javascript é uma linguagem de programação considerada não compilada, ou interpretada, ou seja, o navegador executa diretamente os scripts linha-a-linha.

O javascript é "case sensitive", as letras maiúsculas e minúsculas fazem diferença.

Como adicionar scripts

Os scripts podem ser adicionados em um documento HTML, dentro tag <script> </script> e localizada acima da tag de fechamento </body>.

O Javascript também pode ser adicionado em um documento separado e anexado acima da tag de fechamento </body>:

<script src="scripts/my-scripts.js"> </script>

Tipos de Dados / Valores

Segundo o padrão ECMAScript 6 mais recente, são sete os tipos de dados, sendo seis deles chamados de dados primitivos.

  • Number (Número): Números inteiros ou com casas decimais.
  • String: Cadeia/agrupamento de caracteres (letras, números, símbolos, espaços, etc). Devem estar entre aspas simples, duplas ou crases.
  • Boolean (Booleano): Tipo de dado com apenas dois valores possíveis: verdadeiro (true) ou falso (false).
  • null (Nulo): Aparecendo sempre em letras minúsculas, representa um valor inexistente.
  • undefined (Indefinido): Também escrito em letras minúsculas, aparece quando uma variável não recebeu um valor.
  • Symbol (Símbolo): É um recurso mais recente no ECMAScript, utilizado em códigos mais avançados. Ele armazena um valor único e imutável.
  • Object (Objeto): Diferente dos tipos de dados anteriores, o object não é um tipo primitivo. Ele pode ser descrito como uma coleção de dados e propriedades.

Comentários

Para organização do código, é possível se deixar notas textuais. Os comentários são informações explicativas a respeito da construção e organização, alguma função utilizada, instruções ou dicas, funcionamento dos blocos ou regras aplicadas, seja para o autor do código entendê-lo quando retornar após algum tempo aquele arquivo, quanto para códigos colaborativos.

Essas referências têm somente fim informativo e não são interpretadas pelo navegador.

  • Comentários linha única: são aplicados utilizando duas barras //. Esse tipo de comentário pode ser adicionado em uma nova linha ou após um código funcional, mas nunca antes dele, uma vez que tudo depois de // é lido como um comentário e ele não possui fechamento, terminando somente na quebra de linha.

    window.alert("Alerta!"); // Dispara uma mensagem de alerta

  • Comentários de múltiplas linhas: são criados a partir de uma barra e um asterisco /* e possuem fechamento */. Podem ser inseridas várias quebras de linha nesse tipo de comentário, uma vez que ele só é finalizado com o fechamento. Tudo entre /* e */ é um comentário, portanto também pode ser adicionado no meio das linhas de códigos.

    window.alert(/*Dispara uma mensagem de alerta*/ "Alerta!");

Console

O console é um painel onde os comandos ou expressões são executados e interpretados conforme são digitados.

No JavaScript, console trata-se de um objeto. Pode ser embutido a ele, principalmente, o método .log().

O método console.log() serve para exibir (imprimir ou registrar) uma mensagem, variável, parâmetro, etc, dentro da ferramenta do desenvolvedor. Esse método é muito utilizado com o fim de teste e depuração (localização e correção de erros) do código.

console.log("Essa mensagem será exibida!");
