Acessando valores

Os valores podem ser acessados através de inputs.

<input type="number" id="iNumb1">
<input type="number" id="iNumb2">

Então os valores então são acessados via javascript.

var numero1 = window.document.
querySelector("input#iNumb1");  

  var numero2 = window.document.
querySelector("input#iNumb2");

Adição

var soma = numero1 + numero2;

Subtração

var subtracao = numero1 - numero2;

Alguns inputs ou entradas de valores, acabam convertendo os números em strings (ou strings em números). Nessas ocasiões, é possível converte-los.

Convertendo Valor para String

var strin1 = String(input1.value);
ou var strin2 = input2.value.toString();

Cálculo:

var soma = strin1 + strin2;

+

Resultado da Soma

Convertendo Valor para Número

var num1 = Number.parseInt(input1.value)
Converte a string em número inteiro.

var num2 = Number.parseFloat(input2.value)
Converte a string número real.

var num1 = Number(input1.value);  
  var num2 = Number(input2.value);
Deixa o programa decidir qual tipo de número é.

Cálculo:

var soma = num1 + num2;

+

Resultado da Soma
